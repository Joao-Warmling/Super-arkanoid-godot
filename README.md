▶️ Como executar o projeto
1. Instalar a Godot
Baixe a engine no site oficial:
https://godotengine.org/download

2. Clonar o repositório
git clone https://github.com/Joao-Warmling/Super-arkanoid-godot.git

4. Abrir o projeto
Abra a Godot
Clique em Import
Selecione o arquivo:
project.godot
Clique em Import & Edit

4. Rodar o jogo
Pressione:
F5
ou clique em Run Project dentro da Godot.

🎮 Controles
Setas ou A/D → mover a plataforma
Espaço → iniciar a bola
n → pular de fase

📌 Sobre o projeto
Este projeto foi desenvolvido para estudo e prática de desenvolvimento de jogos 2D utilizando Godot, incluindo sistemas de colisão, física básica e gerenciamento de cenas.
